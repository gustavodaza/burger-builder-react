## React Burger Builder App

This project was bootstrapped with [Create React App](https://github.com/facebookincubator/create-react-app).


### What I used?

- React 16
- React Router
- React-Thunk
- React Sagas
- Redux
- Axios
- Jest / Enzyme
- Webpack
- Babel
- HTML5
- CSS3 w/ Mobile First
- FireBase for Database and Hosting

## How Can I Start?

<pre>npm install</pre>
then
<pre>npm start</pre>
